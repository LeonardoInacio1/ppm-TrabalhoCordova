1 - Instalar o Android Studio devidamente instalado e configuradas as variáveis de ambiente.
2 - Instalar o node.js, o executável de instalação se encontra dentro da pasta instaladores do repositório.
3 - Instalar o Git, o executável de instalação se encontra dentro da pasta instaladores do repositório.
4 - Com o node e o Git devidamente instalados, abrir o prompt de comando, e executar o comando “npm install -g cordova”, 
	após instalado, executar o comando “cordova --version” para verificar a versão.
5 - Instalar o Gradle, o executável de instalação se encontra dentro da pasta instaladores do repositório.
6 - Instalar os seguintes programas Java jdk, Android sdk, Android target.
7 - Abrir o prompt e navegar até a pasta do projeto e executar o comando “cordova requirements” para verificar se cumpre todos os requisitos.
6 - Com tudo devidamente instalado e configurado, abrir o Android Studio, configurar um novo dispositivo de emulação (Nexus 5X API 29), image: Q .
8 - Com o dispositivo pronto para emular, abrir a pasta do projeto no Android Studio, e emular o projeto.
9 - Abrir o prompt e navegar até a pasta do projeto e executar o comando “cordova emulate android”, o emulador deverá 
	reconhecer a execução do aplicativo e abrir no Nexus 5X;
